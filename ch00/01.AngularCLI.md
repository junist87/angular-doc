# Create Project

## Angular CLI 이용하기

* Angular CLI 설치하기
> ~~~
> npm install -g @angular/cli
> ~~~

* Angular Project 생성하기
> ~~~
> ng new [Project 이름]
> ~~~

* Code Generator
> ~~~
> ng g [또는 generate] [생성할 템플릿] [템플릿 이름] 
> ~~~
> * 템플릿 종류  
>> 
>> 1. component  
>> 2. directive
>> 3. pipe
>> 4. service  
>> 5. class
>> 6. interface
>> 7. enum
>> 8. module
>>
